# Папка для фонов

Здесь хранятся файлы фонов для HTML-страниц.

## Поддерживаемые форматы:
- `.jpg` / `.jpeg`
- `.png`
- `.webp`
- `.gif` (статичные кадры)

## Использование:

1. **Для локальной разработки:**
   - Просто загрузите файлы фонов в эту папку
   - Файлы будут доступны по пути `/static/backgrounds/имя_файла`

2. **Для Vercel:**
   - Файлы автоматически деплоятся вместе с проектом
   - Или используйте Vercel CLI для загрузки:
     ```bash
     vercel --prod
     ```
   - Файлы будут доступны по тому же пути

## Переименование файлов:
- Используйте понятные имена: `background1.jpg`, `nature.jpg`, `gradient.png` и т.д.
- Избегайте пробелов в именах (используйте `_` или `-`)
